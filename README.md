# Stitch Aether — Futuristic Luxury E-Commerce Platform

A premium futuristic e-commerce web application built with modern frontend technologies, glassmorphism aesthetics, and scalable SaaS-inspired architecture.

## ✨ Features

* Futuristic luxury UI design
* Dark mode glassmorphism interface
* Fully responsive layouts
* Modern SaaS-style dashboard
* Dynamic product browsing
* Product detail pages
* Wishlist functionality
* Shopping cart system
* Smooth Framer Motion animations
* Reusable component architecture
* Zustand state management
* TypeScript support
* Next.js App Router architecture

# ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/Sidiiq5/stitch-aether-commerce.git
```

Navigate into the project:

```bash
cd stitch-aether-commerce
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:3000
```

---

# 🛠 Build Production Version

```bash
npm run build
```

Start production server:

```bash
npm start
```
